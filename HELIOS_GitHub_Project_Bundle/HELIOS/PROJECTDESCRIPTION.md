
# PROJECTDESCRIPTION

## Canonical project
HELIOS is the single canonical project root.
MONADO is the runtime engine nested inside HELIOS.

## Merge doctrine
Nothing important is discarded.
When contradictions exist:
1. Keep the stricter security rule.
2. Keep the cleaner install / execution flow.
3. Keep the lower-overhead option for GAMING and MUSIC.
4. Keep Bitwarden as the live credential source of truth.
5. Keep the VHDX helper vault as a local encrypted helper, not a Bitwarden replacement.
6. Keep Azure / Entra / Intune / Purview / Key Vault as future policy and governance edge rather than the first vault.
7. Keep HELIOS as the only official root moving forward.

## Canonical build chain
SOLGATE -> LUMEN -> MONADO -> HELIOS Control Center -> AEGIS / SENTINEL / ORCHESTRA
