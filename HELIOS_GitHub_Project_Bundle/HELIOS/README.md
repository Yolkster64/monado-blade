
# HELIOS

HELIOS is the canonical project root for a local-first, security-first, performance-optimized Windows workstation architecture.

## Core chain
- SOLGATE = USB builder / offline installer path
- LUMEN = first-run setup UI
- MONADO = runtime / execution engine
- HELIOS Control Center = ongoing command center
- AEGIS = lightweight analyzer / detection
- SENTINEL = enforcement / security
- ORCHESTRA = future supervised AI coordination

## Intent
This bundle is designed to be uploaded to GitHub as the canonical starting point for the full HELIOS platform.
It preserves older Helios package ideas while carrying forward the newer HELIOS / MONADO architecture.
