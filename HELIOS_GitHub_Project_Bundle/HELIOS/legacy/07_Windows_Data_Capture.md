# Windows Data Capture

## What you can safely capture into files
- machine name
- local users
- installed apps
- signed drivers
- Wi-Fi profile names
- USB / disk inventory
- BitLocker status and protector info

## What you cannot safely auto-capture
- hidden Windows passwords
- authenticator seeds
- secrets apps do not export

## Use captured data for
- Add-New CSV population
- recovery notes
- reset planning
- admin audits
