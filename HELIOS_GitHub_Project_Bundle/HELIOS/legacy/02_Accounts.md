# Accounts

## Roles
- **ADMIN**: local admin only; installs, system changes, audits
- **WORK**: daily secure environment; Bitwarden lives here
- **GAMING**: performance-first, minimal background
- **DEV**: testing / coding / sandboxing
- **MUSIC**: low-latency production

## Recommended rights
- ADMIN = Administrator
- WORK / GAMING / DEV / MUSIC = Standard User
