# Bitwarden

## Role
Bitwarden is the live credential source of truth.

## Structure
Use folders or collections like:
- HELIOS/EMAIL
- HELIOS/MICROSOFT
- HELIOS/SYSTEM
- HELIOS/VENDORS
- HELIOS/DEV
- HELIOS/VAULT_CORE
- HELIOS/ADD_LATER

## Ongoing rule
- Browser stores nothing long-term.
- Bitwarden captures new credentials through save prompts.
- Bitwarden updates changed credentials through update prompts.
- Anything not captured goes into Add-New CSV or a secure note.

## Good core entries
- Gmail / Outlook / Microsoft
- Bitwarden
- Razer / NVIDIA / Malwarebytes
- eero / router
- Windows account references
