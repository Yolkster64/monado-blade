# Google Drive Layout

## Good things to sync
- the Helios folder
- markdown/system docs
- inventories
- non-secret setup notes
- screenshots and diagrams

## Things to avoid syncing long-term
- plaintext password export CSVs
- raw Bitwarden exports
- temporary import files

## Suggested Drive layout
Google Drive/
  Helios/
    docs/
    scripts/
    inventories/
    templates/
    exports_temp/   (clean this after import)
