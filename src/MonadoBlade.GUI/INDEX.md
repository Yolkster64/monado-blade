## 🎨 Visual Design Quick Reference

**Monado Blade Core:**
- Spinning circle: 60-300 RPM (adaptive to progress)
- Gear teeth: 8 segments, counter-rotating
- Center crystal: Pulsing (50-100% intensity)
- Colors: Cyan → Blue → Gold → Orange → Green
- Glow: 5-30px radius (progress-dependent)

**Progress Journey:**
1. Begin (60 RPM, cyan)
2. Configure (120 RPM, light blue)
3. Select (180 RPM, cyan-blue)
4. Prepare (200 RPM, blue) - **50% MILESTONE**
5. Verify (240 RPM, gold)
6. Download (270 RPM, orange)
7. Verify (280 RPM, gold)
8. Success (300 RPM, green-gold celebration)

**Kanji Characters:**
始 → 設定 → 選択 → 準備 → 検証 → ダウンロード → 成功
